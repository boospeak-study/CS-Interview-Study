# About

- **CS 지식을 학습하고 머릿속 지식을 말로 설명할 수 있는 능력을 기릅니다.**
- 스터디는 **온라인 3회, 오프라인 1회** 비율로 진행합니다.
- 온라인 스터디에서는 **전원이 카메라를 켠** 상태로 참여합니다.
- **매주 KPT 회고**를 진행합니다. 이를 바탕으로 스터디 진행 사항을 개선해 나갑니다.
